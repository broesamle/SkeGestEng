### Transcript Integration -- Build The Corpus
==============================================

* Transkripte (Word/`*.RTF`)
* Video-Annotationen (Adobe PRemiere/`*.PRPRJ`)
in eine XML-Datei für die Sequentiellen Daten -- d.h. Text, Zeitmarken, Gesten-Intervalle etc...
